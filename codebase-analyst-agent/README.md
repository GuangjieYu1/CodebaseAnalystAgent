# Codebase Analyst Agent

一个面向本地代码仓库的轻量 Agent 原型，用于自动扫描项目结构、识别技术栈、定位关键文件，并回答基础代码理解问题。

## 1. 项目目标

本项目的目标是验证一个最小可行的代码分析 Agent：

- 读取本地代码仓库
- 识别项目技术栈
- 推荐关键阅读文件
- 总结指定文件职责
- 回答简单的代码理解问题

这个项目不追求“自动改代码”，而是先把“读代码 -> 组织上下文 -> 输出分析结果”的闭环打通。

## 2. 适用场景

- 快速接手一个不熟悉的项目
- 理解某个模块的大致职责
- 确定某个功能应该优先看哪些文件
- 给后续代码助手 / 测试生成 Agent 提供基础上下文能力

## 3. 当前功能（P0）

- 扫描项目目录结构
- 检测技术栈（如 Maven / Gradle / Spring Boot）
- 推荐关键文件
- 摘要单个文件
- 基于扫描结果回答简单问题

## 4. 非目标

- 自动修改代码
- 自动生成补丁
- 复杂多 Agent 编排
- 全量代码索引系统
- IDE 插件集成
